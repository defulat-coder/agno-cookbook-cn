# Accuracy Eval Cookbooks（准确性评估示例集）

准确性示例评估响应结果与预期输出的匹配程度。

## 文件列表

- `accuracy_basic.py` - 同步与异步计算器准确性评估。
- `accuracy_9_11_bigger_or_9_99.py` - 小数值的数值比较准确性评估。
- `accuracy_team.py` - Team（团队）语言路由准确性评估。
- `accuracy_with_given_answer.py` - 对已提供输出字符串进行准确性评分。
- `accuracy_with_tools.py` - 使用工具的 Agent（代理）的准确性评估。
- `db_logging.py` - 将准确性评估结果记录到 PostgreSQL。
- `evaluator_agent.py` - 使用自定义 Evaluator（评估器）Agent 进行准确性评估。
