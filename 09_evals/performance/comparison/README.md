# Performance Comparison Cookbooks（性能对比示例集）

这些基准测试对比了非 Agno 框架的 Agent（代理）实例化模式。

## 文件列表

- `autogen_instantiation.py` - AutoGen 助手实例化基准测试。
- `crewai_instantiation.py` - CrewAI Agent（代理）实例化基准测试。
- `langgraph_instantiation.py` - LangGraph react-agent 实例化基准测试。
- `openai_agents_instantiation.py` - OpenAI Agents SDK 实例化基准测试。
- `pydantic_ai_instantiation.py` - PydanticAI Agent（代理）实例化基准测试。
- `smolagents_instantiation.py` - Smolagents 工具调用 Agent（代理）实例化基准测试。
