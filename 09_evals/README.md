# Evals Cookbook（评估示例集）

本目录包含 Agno 评估模式的可运行示例。

## 目录概览

- `accuracy/` - Agent（代理）和 Team（团队）的准确性评估示例。
- `agent_as_judge/` - 基于模型评分的 LLM-as-judge（LLM 作为裁判）评估示例，含评分与钩子。
- `performance/` - 运行时与内存性能基准测试示例。
- `performance/comparison/` - 非 Agno 框架的实例化基准对比。
- `reliability/` - 工具调用可靠性评估示例。
- `reliability/single_tool_calls/` - 单次预期工具调用的可靠性示例。
- `reliability/multiple_tool_calls/` - 多工具工作流的可靠性示例。
- `reliability/team/` - Team（团队）工具调用流程的可靠性示例。

## 根目录文件

- `RESTRUCTURE_PLAN.md` - 详细的重构计划与文件处理说明。
- `RESTRUCTURE_PROMPT.md` - 重构任务的实现提示。
