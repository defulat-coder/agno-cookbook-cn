# Reliability Eval Cookbooks（可靠性评估示例集）

可靠性示例验证预期的工具调用是否被正确执行。

## 文件列表

- `db_logging.py` - 带 PostgreSQL 日志记录的可靠性评估。
- `reliability_async.py` - 异步可靠性评估流程。
- `single_tool_calls/calculator.py` - 单次预期计算器工具调用的可靠性验证。
- `multiple_tool_calls/calculator.py` - 多工具计算器调用的可靠性验证。
- `team/ai_news.py` - Team（团队）委托与网络搜索工具调用的可靠性验证。
