# Cookbook Python 风格指南

本指南规范可运行的 cookbook `.py` 示例文件。

## 核心模式

1. 文件顶部的模块文档字符串：
- 本示例演示的内容
- 关键概念
- 可尝试的提示词或输入

2. 使用横幅注释划分段落：
- 配置段落（存储、工具、知识库、数据模型等按需添加）
- 指令段落
- `Create ...` 创建段落
- `Run ...` 运行段落
- 可选的 `More Examples` 更多示例段落

3. 主执行入口：
- `if __name__ == "__main__":`
- 将可运行的演示步骤放在此代码块中

4. cookbook Python 文件中不使用 emoji 字符。
