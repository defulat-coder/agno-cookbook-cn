# Knowledge Cookbook

AgentOS 中 `knowledge` 的示例。

## 文件
- `agentos_excel_analyst.py` — AgentOS Excel 分析器。
- `agentos_knowledge.py` — AgentOS 知识库（同步和异步）。
- `agno_docs_agent.py` — Agno 文档 Agent。

## 前置条件
- 使用 `direnv allow` 加载环境变量（需要 `.envrc` 文件）。
- 使用 `.venvs/demo/bin/python <path-to-file>.py` 运行示例。
- 某些示例需要本地服务（例如 Postgres、Redis、Slack 或 MCP 服务器）。
