# memory（记忆）

memory 中团队工作流的示例。

## 前置条件

- 通过 direnv allow 加载环境变量（例如 OPENAI_API_KEY）。
- 使用 .venvs/demo/bin/python 运行手册示例。
- 某些示例需要额外的服务（例如 PostgreSQL、LanceDB 或 Infinity 服务器），如文件文档字符串中所述。

## 文件说明

- 01_team_with_memory_manager.py - 演示带内存管理器的团队。
- 02_team_with_agentic_memory.py - 演示带 Agentic 记忆的团队。
- learning_machine.py - 演示学习机器。
