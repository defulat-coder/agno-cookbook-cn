# multimodal（多模态）

multimodal 中团队工作流的示例。

## 前置条件

- 通过 direnv allow 加载环境变量（例如 OPENAI_API_KEY）。
- 使用 .venvs/demo/bin/python 运行手册示例。
- 某些示例需要额外的服务（例如 PostgreSQL、LanceDB 或 Infinity 服务器），如文件文档字符串中所述。

## 文件说明

- audio_sentiment_analysis.py - 演示音频情感分析。
- audio_to_text.py - 演示音频转文本。
- generate_image_with_team.py - 演示团队生成图像。
- image_to_image_transformation.py - 演示图像到图像转换。
- image_to_structured_output.py - 演示图像到结构化输出。
- image_to_text.py - 演示图像转文本。
- media_input_for_tool.py - 演示工具的媒体输入。
- video_caption_generation.py - 演示视频字幕生成。
