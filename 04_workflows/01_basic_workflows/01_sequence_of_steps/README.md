# 04_workflows/01_basic_workflows/01_sequence_of_steps

## 范围
cookbook/04_workflows/01_basic_workflows/01_sequence_of_steps 目录下的可运行 Workflow（工作流）示例

## 文件
- sequence_of_steps.py: 演示步骤序列
- sequence_with_functions.py: 演示带函数的序列
- workflow_using_steps.py: 演示使用步骤的工作流
- workflow_using_steps_nested.py: 演示使用嵌套步骤的工作流
- workflow_with_file_input.py: 演示带文件输入的工作流
- workflow_with_session_metrics.py: 演示带会话指标的工作流

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
