# 04_workflows/02_conditional_execution

## 范围
cookbook/04_workflows/02_conditional_execution 目录下的可运行 Workflow（工作流）示例

## 文件
- condition_basic.py: 演示基础条件
- condition_with_else.py: 演示带 else 的条件
- condition_with_list.py: 演示带列表的条件
- condition_with_parallel.py: 演示带并行的条件

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
