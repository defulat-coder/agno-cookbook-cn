# 04_workflows/07_cel_expressions/condition

## 范围
cookbook/04_workflows/07_cel_expressions/condition 目录下的可运行 Workflow（工作流）示例

## 文件
- cel_additional_data.py: 演示 CEL 附加数据
- cel_basic.py: 演示 CEL 基础
- cel_previous_step.py: 演示 CEL 前序步骤
- cel_previous_step_outputs.py: 演示 CEL 前序步骤输出
- cel_session_state.py: 演示 CEL 会话状态

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
- CEL 示例需要安装 cel-python
