# 04_workflows/07_cel_expressions/router

## 范围
cookbook/04_workflows/07_cel_expressions/router 目录下的可运行 Workflow（工作流）示例

## 文件
- cel_additional_data_route.py: 演示 CEL 附加数据路由
- cel_previous_step_route.py: 演示 CEL 前序步骤路由
- cel_session_state_route.py: 演示 CEL 会话状态路由
- cel_ternary.py: 演示 CEL 三元运算
- cel_using_step_choices.py: 演示 CEL 使用步骤选项

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
- CEL 示例需要安装 cel-python
