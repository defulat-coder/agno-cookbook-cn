# 04_workflows/07_cel_expressions/loop

## 范围
cookbook/04_workflows/07_cel_expressions/loop 目录下的可运行 Workflow（工作流）示例

## 文件
- cel_compound_exit.py: 演示 CEL 复合退出
- cel_content_keyword.py: 演示 CEL 内容关键字
- cel_iteration_limit.py: 演示 CEL 迭代限制
- cel_step_outputs_check.py: 演示 CEL 步骤输出检查

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
- CEL 示例需要安装 cel-python
