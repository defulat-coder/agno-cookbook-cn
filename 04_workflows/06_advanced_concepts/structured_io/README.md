# 04_workflows/06_advanced_concepts/structured_io

## 范围
cookbook/04_workflows/06_advanced_concepts/structured_io 目录下的可运行 Workflow（工作流）示例

## 文件
- image_input.py: 演示图片输入
- input_schema.py: 演示输入模式
- pydantic_input.py: 演示 Pydantic 输入
- structured_io_agent.py: 演示结构化 IO 的 Agent
- structured_io_function.py: 演示结构化 IO 的函数
- structured_io_team.py: 演示结构化 IO 的 Team（团队）

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
