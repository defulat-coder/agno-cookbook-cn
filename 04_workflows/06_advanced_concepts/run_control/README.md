# 04_workflows/06_advanced_concepts/run_control

## 范围
cookbook/04_workflows/06_advanced_concepts/run_control 目录下的可运行 Workflow（工作流）示例

## 文件
- cancel_run.py: 演示取消运行
- deep_copy.py: 演示深拷贝
- event_storage.py: 演示事件存储
- executor_events.py: 演示执行器事件
- metrics.py: 演示指标
- remote_workflow.py: 演示远程工作流
- workflow_cli.py: 演示工作流命令行接口
- workflow_serialization.py: 演示工作流序列化

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
- 某些示例需要本地 AgentOS 服务（参见文件头部的服务器 URL）
