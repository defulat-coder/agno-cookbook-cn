# 04_workflows/06_advanced_concepts/background_execution

## 范围
cookbook/04_workflows/06_advanced_concepts/background_execution 目录下的可运行 Workflow（工作流）示例

## 文件
- background_poll.py: 演示后台轮询
- websocket_client.py: 演示 WebSocket 客户端
- websocket_server.py: 演示 WebSocket 服务器

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
- 某些示例需要本地 AgentOS 服务（参见文件头部的服务器 URL）
