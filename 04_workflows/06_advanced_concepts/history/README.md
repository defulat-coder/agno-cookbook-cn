# 04_workflows/06_advanced_concepts/history

## 范围
cookbook/04_workflows/06_advanced_concepts/history 目录下的可运行 Workflow（工作流）示例

## 文件
- continuous_execution.py: 演示连续执行
- history_in_function.py: 演示函数中的历史记录
- intent_routing_with_history.py: 演示带历史记录的意图路由
- step_history.py: 演示步骤历史记录

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
