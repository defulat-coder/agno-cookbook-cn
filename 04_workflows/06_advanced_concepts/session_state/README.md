# 04_workflows/06_advanced_concepts/session_state

## 范围
cookbook/04_workflows/06_advanced_concepts/session_state 目录下的可运行 Workflow（工作流）示例

## 文件
- rename_session.py: 演示重命名会话
- state_in_condition.py: 演示条件中的状态
- state_in_function.py: 演示函数中的状态
- state_in_router.py: 演示路由器中的状态
- state_with_agent.py: 演示 Agent 的状态
- state_with_team.py: 演示 Team（团队）的状态

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
