# 04_workflows/06_advanced_concepts/early_stopping

## 范围
cookbook/04_workflows/06_advanced_concepts/early_stopping 目录下的可运行 Workflow（工作流）示例

## 文件
- early_stop_basic.py: 演示基础提前停止
- early_stop_condition.py: 演示条件提前停止
- early_stop_loop.py: 演示循环提前停止
- early_stop_parallel.py: 演示并行提前停止

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
