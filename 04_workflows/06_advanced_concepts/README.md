# 04_workflows/06_advanced_concepts

## 范围
cookbook/04_workflows/06_advanced_concepts 目录下的可运行 Workflow（工作流）示例

## 子目录
- background_execution/: 后台执行的附加示例
- early_stopping/: 提前停止的附加示例
- guardrails/: 护栏的附加示例
- history/: 历史记录的附加示例
- long_running/: 长时间运行的附加示例
- previous_step_outputs/: 前序步骤输出的附加示例
- run_control/: 运行控制的附加示例
- session_state/: 会话状态的附加示例
- structured_io/: 结构化 IO 的附加示例
- tools/: 工具的附加示例
- workflow_agent/: Workflow Agent 的附加示例

## 先决条件
- 激活演示环境：.venvs/demo/bin/python
- 使用 direnv allow 加载 API 密钥（需要本地 .envrc 文件）
